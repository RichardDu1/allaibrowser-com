## Overview
This workflow connects your inventory data (e.g., from Shopify or a spreadsheet) to an AI writing tool like Jasper, automatically generating unique, SEO-friendly product descriptions. You'll set up a trigger (new product added) and an action (generate description), then review and publish.

## Prerequisites
- A product inventory source (e.g., Shopify, WooCommerce, or Google Sheets)
- A Jasper AI account (free trial available)
- A Zapier account (free tier includes 100 tasks/month)

## Step 1: Prepare Your Inventory Data
Ensure your inventory data includes fields like: Product Name, Category, Key Features, Price, and Target Keywords. If using Google Sheets, create columns for these fields.

## Step 2: Set Up Zapier Trigger
1. Log in to Zapier and create a new Zap.
2. Choose your inventory source as the trigger app (e.g., Shopify: New Product, or Google Sheets: New Row).
3. Connect your account and select the relevant spreadsheet or product list.
4. Test the trigger to ensure Zapier can pull sample data.

## Step 3: Configure Jasper AI Action
1. Add a new action step and select Jasper AI.
2. Choose the action: "Generate Content".
3. Connect your Jasper account (API key from Jasper settings).
4. Map the inventory fields to Jasper's template variables. For example:
   - Product Name → `{{product_name}}`
   - Key Features → `{{features}}`
   - Target Keyword → `{{keyword}}`
5. Write a prompt template like: "Write a compelling product description for {{product_name}}. Highlight these features: {{features}}. Use the keyword {{keyword}} naturally. Keep it under 150 words."
6. Set tone and length preferences (e.g., persuasive, 100-150 words).
7. Test the action to see a sample description.

## Step 4: Add Output Destination
Add a final action to store the generated description. Options:
- Update the same spreadsheet in a new column (Google Sheets: Update Cell)
- Send to your CMS (e.g., Shopify: Update Product)
- Email the description to yourself for review

## Step 5: Review and Refine
Check the first few generated descriptions. Adjust the prompt template if needed (e.g., add brand voice guidelines). You can also add a human review step before publishing.

## Step 6: Automate and Scale
Turn on the Zap. Every new product added to your inventory will automatically receive a draft description. You can also run the Zap on existing products by triggering it manually.

## Tips for Best Results
- Include specific data points (dimensions, materials) for accuracy.
- Use Jasper's "Boss Mode" for more control over output.
- Regularly update your keyword list to match SEO trends.
- Combine with A/B testing to optimize conversion rates.

## Troubleshooting
- If descriptions are too generic, add more specific fields to your inventory data.
- If Zapier fails, check API limits and re-authenticate accounts.
- For large inventories, consider batching or using a dedicated integration tool.
