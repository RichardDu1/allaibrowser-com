## Overview
This workflow enables a global social media manager to take a single English brief and generate culturally adapted posts in multiple languages. It leverages AI for translation, localization, and visual adaptation, then schedules posts across platforms.

## Prerequisites
- A single English brief (e.g., product launch, campaign theme)
- Access to the tools listed below
- Brand guidelines (tone, voice, visual assets)

## Step-by-Step Guide

### Step 1: Prepare the English Brief
Write a comprehensive brief including:
- Key message and call-to-action
- Target audience per market
- Cultural nuances or taboos to avoid
- Hashtags and keywords
- Visual references

### Step 2: Translate and Localize with DeepL
Use DeepL to translate the brief into target languages (e.g., Spanish, French, Japanese).
- Go to [DeepL](https://www.deepl.com) and paste the brief.
- Select target language and translate.
- Review for accuracy; adjust idioms or phrases that don't localize well.
- Save each translation as a separate document.

### Step 3: Adapt Tone and Culture with ChatGPT
Use ChatGPT to refine translations for social media tone and cultural relevance.
- Prompt example: "Adapt the following Spanish translation for a Mexican audience on Instagram: [text]. Make it conversational and use local slang where appropriate."
- Repeat for each market.
- Ensure brand voice consistency by providing brand guidelines in the prompt.

### Step 4: Create Visuals with Canva
Use Canva to adapt visuals for each market.
- Open the English design template in Canva.
- Replace text with localized versions.
- Swap images if needed (e.g., culturally appropriate models, landmarks).
- Use Canva's Magic Write to generate alt text in each language.
- Export as PNG/JPEG for each platform.

### Step 5: Schedule Posts with Hootsuite
Use Hootsuite to schedule posts across markets.
- Create separate Hootsuite streams for each language/region.
- Upload localized images and captions.
- Set optimal posting times based on time zones.
- Use Hootsuite's approval workflow for compliance review.

### Step 6: Monitor and Iterate
- Track engagement per market using Hootsuite analytics.
- Collect feedback from local teams.
- Update the English brief with learnings for future campaigns.

## Best Practices
- Always have a native speaker review AI-generated content.
- Maintain a glossary of brand terms in each language.
- Use consistent hashtags across markets but add local trending tags.
- Test posts on a small audience before full rollout.

## Troubleshooting
- **Translation too literal?** Use ChatGPT with a prompt to make it more natural.
- **Visuals not resonating?** Research local visual preferences on Pinterest or Instagram.
- **Scheduling conflicts?** Use Hootsuite's bulk upload feature for multiple posts.
